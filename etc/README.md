# Configuration

## Installation

Install via:
```bash
make install
```

Or copy `conf` to `~/.config/i3blocks-modules/conf` in order to set global module options.


## Configuration

You can define global default colors for all modules to fit your theme:

```
# Set your default colors
color_def="#666666"
color_good="#88b090"
color_warn="#ccdc90"
color_crit="#e89393"
color_info="#FCE94F"
```
